# Project Brief: Riviera Residence

## Overview
Riviera Residence is a cinematic landing page for a luxury residential project on the Bosphorus in Istanbul. The site features 8 sections with dark backgrounds, gold accents, and very slow cinematic animations.

## Core Requirements
- 8 distinct sections: Hero, Project Summary, Apartment Types, Interior Gallery, Location, Facilities, Contact Form, Footer
- Dark theme with gold accents (#C9A96E)
- Typography: Playfair Display (headings) + Inter (body)
- Very slow cinematic animations (1s+)
- Responsive design (mobile-first)
- No build errors

## Target Audience
- High-net-worth individuals
- Luxury real estate investors
- International buyers interested in Istanbul property

## Key Metrics
- 45 floors, 120 apartments, 95% Bosphorus view, 2027 delivery
